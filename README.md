## üöÄGetting Started

1. Clone the project

```bash
git clone https://github.com/Skipi0007/Playwright-tests-example.git
```

2. Install project dependencies

```bash
npm ci --legacy-peer-deps
#or same with
yarn
```

3. Run the development server in project root path:

```bash
npm run dev
# or
yarn dev
```

4. Run playwright tests:

```bash
# withou/with ui
npm run tests
npm run tests --ui
# or
yarn tests
yarn tests --ui
```
